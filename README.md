# ⚖️ Legal Document Simplifier

An AI-powered tool that simplifies complex Indian legal documents and contracts into plain, easy-to-understand language — with translation support for Tamil and Malayalam to improve accessibility for non-English speakers.

## Features
- Simplifies dense legal jargon into plain language
- Tailored for Indian legal contexts (contracts, agreements, terms)
- Translation support: English → Tamil, English → Malayalam
- Clean web interface for uploading/pasting legal text

## Tech Stack
- **Backend:** Python, Flask
- **AI/NLP:** Gemini API (or specify if different)
- **Translation:** (add library/API used, e.g. Google Translate API)
- **Frontend:** HTML, CSS, JavaScript

## How It Works
1. User uploads or pastes a legal document
2. The system breaks it down and simplifies complex clauses using AI
3. User can optionally translate the simplified output into Tamil or Malayalam
